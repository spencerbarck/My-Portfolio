Very simple Calculator

A java and swing program

This calculator app is pretty self explanatory. It has addition, subtraction, multiplication, and division. I have made a couple of calculator programs over my years of learning programming but I realized I have never actually made one with a GUI. I figured making this user interface could help me learn more about Swing and Grid Layouts.

In the future I might add more functionality to the calculator like square root, parentheses, and exponent. If I really wanted to go back into calculus I could add log but I think for now I will leave my calculator the way it is.
