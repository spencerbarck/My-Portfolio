Find Kaprekar

A java and swing program

The Find Kaprekar app allows the user to enter an integer and find all the Kaprekar numbers that occur before that integer. A Kaprekar number is a number where if you square the number, then split the squared value into two numbers, you can add those numbers and get the original number. For instance 45^2 = 2025 and 20+25=45 therefore 45 is a Kaprekar number.

I decided to give the app a little more functionality. The user can check the Find X Kaprekars box to find the first X Kaprekars starting from 1. I also added a section at the bottom where the user can enter a number and find out if that number is a Kaprekar number.
