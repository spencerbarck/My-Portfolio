The Multiverse Generator

A java and swing program

The Multiverse generator app is an app that creates zero or more multiverses with random names, then each multiverse creates zero or more galaxies which create zero or more solar systems which generate zero or more planets. I decided to make this app because of my love for both comic books and space travel. In comic books the word multiverse refers to a collection of universes and I figured I could create a sort of russian nesting doll using this concept.

The first thing you will be greeted with when you use the app is a login page. To login you must create a profile using a username, password and email, or you can simply use my login username:spencer, password:mango. Once you are logged in you can press the Make a Multiverse button and create a multiverse with one universe. To create multiple universes at one time click settings. In settings you can change the amount of universes, galaxies, solar systems, and planets. You can also make the number random and make it so no solar system has zero planets, no galaxy has zero solar systems and so forth. The reset button allows you to start over and the profile button lets you look at your user profile.
