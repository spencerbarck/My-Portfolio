Duplicate Finder

A java and swing program

The duplicate counter is an app that allows the user to enter an array of numbers separated by commas and finds what numbers are repeated. I tried to make this user interface have a simple design with bright colors that look good on the eye.

In the future if I wanted to expand on this app I could add the same functionality for characters and I could make a setting page. In the setting page I could give the user an option to find out how many times each number appears and make it so the user could change the colors of the UI.
