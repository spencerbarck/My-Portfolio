Rand Int App

A java and swing program

The Rand Int app lets the user enter two integers and find a random number between the two. This app was very easy to make and the only problem I had was making sure that the random number can not be one of the two integers the user entered.

I can see myself using this app in the future when I want a random number. It is fun to make apps that I use in my spare time even if those apps are simple.
