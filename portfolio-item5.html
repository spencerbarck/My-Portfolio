Clump Finder

A java and swing program

The Clump Finder app allows the user to enter an array of numbers separated by commas and finds the number clumps which are groups of identical numbers. The app outputs the clumps shows what numbers are in the clump and how many. For example if the user enters 1,1,2,3,3 the app will output Clump 1 {1,1} Clump 2 {3,3} the clump number is simply the order in which the clumps appeared in the string.

I made the app after solving a programming exercise where you were supposed to count the number of clumps in a string. I decided to make things a little more difficult for myself by also keeping track of the clumps themselves. I also made this entire app using only arrays and no lists. I enjoyed the challenge of coding this app and the GUI that I made for it.
