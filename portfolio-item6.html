Count Ends In X or Y

A java and swing program

The Count Ends in X or Y app has the user enter an array of characters separated by commas and a string with words in it, the app finds the number of words ending in any of the characters in the array. I made this app after a programming exercise and I decided to make it so the user could enter any number of letters they wanted to search for.

In the future I could make it so the user could read through a file and search for words ending in the characters, this would make it easier to say look through a book and find the number of words ending in g. I could also make the app output each word that it finds similar to my Clump Finder app.
