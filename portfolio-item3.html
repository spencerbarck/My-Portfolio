Same End Finder

A java and swing program

The Same ends app is an app that has the user enter a string and find the longest string that appears on both ends of the string. If the user enters a palindrome like civic the app will output ci. The app works with any string.

This app was made in response to a programming exercise I found online. I decided it would be fun to make a GUI to go with my solution to the exercise.
